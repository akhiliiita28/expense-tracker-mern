# Expense Tracker MERN

A full-stack MERN Expense Tracker application for managing income and expenses with transaction tracking and analytics.

---

## Features

- Add income and expense transactions
- Transaction history
- Expense categorization
- Charts and analytics
- Responsive UI
- Full-stack MERN architecture

---

## Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

---

## Project Structure

Expense Tracker/
├── frontend/
├── backend/

---

## Installation

### Clone Repository

git clone https://github.com/akhiliiita28/expense-tracker-mern.git

---

## Backend Setup

cd backend
npm install
npm start

---

## Frontend Setup

cd frontend
npm install
npm run dev

---

## Environment Variables

Create a .env file inside backend folder and add:

PORT=your_port
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
